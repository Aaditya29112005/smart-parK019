# Smart Parking App

A comprehensive smart parking solution built with React, TypeScript, and Tailwind CSS.

## Features
- **User Dashboard**: View recent parking sessions.
- **Scanner**: Scan QR codes to initiate parking.
- **Vehicle Management**: Register and manage multiple vehicles.
- **Digital Tickets**: View and manage active parking tickets.
- **Payments**: Integrated payment flow simulation.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
