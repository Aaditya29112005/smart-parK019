# Smart Park Backend

RESTful API server for the Smart Park application.

## Features

- Express.js server
- CORS enabled
- Health check endpoint
- Ready for deployment on Render

## API Endpoints

- `GET /` - Health check
- `GET /api/health` - API health status

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm start
```

## Environment Variables

No environment variables required for basic operation.

## Deployment

Configured for deployment on Render via `render.yaml` in the root directory.
